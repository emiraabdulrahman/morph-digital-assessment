# morph-digital-assessment

A responsive web page that displays interactive infographic:
- header
- banner (with parralax scroll effect)
- a timeline of client's past achievements (with Scroll Reveal effect: slide up effect when hover to each cards)
- video banner (using GSAP + scrolltrigger) with video modal button **[kindly refresh the page if video banner section is not appeared ]**

Additional element
- hover effect on buttons
- popup modal after 2/3 seconds (the 3D image using Three.js)

Responsive UI (desktop, tablet & mobile)
Web page URL https://deploy-preview-3--poetic-duckanoo-265bf7.netlify.app/
